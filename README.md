#Chat 2Gis
*Тестовая работа на должность Web-developer 2Gis*
-Node.js server
-WebSocket
-Mocha test

## Requirements
	- nodejs v0.12.7
	- npm v3.3.4
	
## Install
	```
    $ npm install
    $ bower install
    ```

## Run
	```
    $ node server-run.js
    ```
	Open in your browser http://127.0.0.1:8080
## Test
	```
    $ gulp test
    ```
